# Contributing

Thanks for considering a contribution to Konoha Agentic Academy.

Start here:

- `README.md`
- `AGENTS.md`
- `docs/contributing/contribution_guide.md`
- `docs/guides/public_private_boundary.md`

## Contribution rules

Public contributions must be generic, license-safe, and free of private context.

Do not commit:

- secrets or credentials;
- local Allied Village files;
- private literature or converted copyrighted sources;
- personal data;
- internal project context;
- private assets;
- generated outputs that may contain sensitive content.

## Doctrine changes

Doctrine changes require explicit review.

Use:

- `scrolls/doctrine_update_scroll.md`
- `shikamaru/scribe_policy.md`
- `protocols/approval/approval_policy.md`
- `protocols/review/review_policy.md`

Do not hide doctrine changes inside regular documentation updates.
