# Local Model Audit Examples

Examples for v3.0.1 local computer profiling, local model recommendation, Ollama install planning, local model repository audit, patch planning, and documentation patch apply reports.

Authority boundaries:

- Local computer profiles are evidence only.
- Local model recommendations are not permission.
- Ollama install plans do not install Ollama.
- Local model download requires explicit approval and network allowance.
- Local model audit output is evidence only.
- Patch plans require explicit approval.
- Git operations must use the existing v3 beta Git gate.
