# Dry-run Package JSON Examples

Machine-readable dry-run package examples for the runtime validator.

These examples are public, generic, and non-executable. They do not authorize shell execution, filesystem mutation, Git operations, adapter execution, autonomous execution, or private context access.

## Validate an example

```powershell
python tools/runtime_validator/validate_runtime_package.py examples/dry_run_packages/docs_update.runtime_package.json
```
