# Model invocation examples

This folder contains example evidence for the v1.5.0 Real Model Invocation Gate.

Files:

- `model_invocation_contract.example.json`: example contract allowing mock and explicitly gated real providers.
- `mock_model_request_plan.example.json`: mock provider request plan for local smoke tests.
- `live_model_request_plan.example.json`: real-provider request plan example that still requires explicit approval and network flag.
- `model_invocation_gate_report.example.json`: example invocation gate report.

The examples are public and synthetic. They do not contain credentials, private context, API responses, or provider secrets.
