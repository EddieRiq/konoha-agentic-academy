# Code of conduct

Konoha Agentic Academy uses a human community code of conduct and a separate agent conduct policy.

For contributors, see:

- `docs/contributing/code_of_conduct.md`

For agents, see:

- `core/conduct/AGENT_CONDUCT.md`

## Basic expectation

Be clear, respectful, and careful with private information.

Do not post secrets, credentials, personal data, internal project context, private literature, or copyrighted material in issues, discussions, pull requests, examples, or documentation.

## Reporting problems

Use the repository issue tracker for general conduct issues that can be discussed publicly.

Do not use public issues to report secrets, vulnerabilities, or private data exposure. Use the security process described in `SECURITY.md`.
