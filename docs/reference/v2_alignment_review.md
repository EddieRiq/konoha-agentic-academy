# v2.0 Alignment Review

Before v2.0.0, Konoha paused to verify that the project remained aligned with its original intent:

```text
local-first
human-in-the-loop
evidence before action
model output is not permission
memory supports action but does not authorize action
safe gates before sensitive actions
```

## v2.0 decision

v2.0.0 is an integration and mission-closure release, not a visual polish release.

Advanced visuals, sounds, marketplace, local model benchmarking, desktop notifications, and multi-agent background autonomy remain out of scope.
