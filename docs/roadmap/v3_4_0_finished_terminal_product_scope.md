# v3.4.0 Finished Local-First Terminal Product Scope

## Objective

Deliver a coherent installed product that guides an operator from first launch
through supervised mission closure without requiring repository-internal
knowledge.

## Included

- empty-invocation product welcome;
- explicit-token quickstart;
- evidence-based `konoha next`;
- user help using installed `konoha` syntax;
- separate mission and maintainer help;
- close-match recovery for unknown commands;
- quiet annotated-tag installation and upgrade;
- post-install and post-upgrade next actions;
- product experience state and report schemas;
- clean-install smoke for welcome, quickstart and next;
- final user guides and capability boundaries.

## Excluded

- Web UI;
- daemon or background agents;
- autonomous execution;
- automatic approval tokens;
- automatic network access;
- repository cleanup or deletion;
- new model providers or external integrations.

Repository rationalization is intentionally deferred until after the finished
product release.
