# v3.0.1 Beta Patch Scope

v3.0.1 is the first beta patch after v3.0.0.

Scope:

- WSL/local computer profile.
- Local model recommendation.
- Ollama install plan.
- Approved Ollama model download.
- Local model repository consistency audit.
- README/index mismatch detection.
- Patch plan generation.
- Approved documentation patch apply.
- Existing beta Git gate for add/commit/push.

Non-goals:

- Full autonomy.
- Remote model invocation.
- Unapproved command execution.
- Secret/private context access.
- Automatic Git push without approval.
- Replacing Jounin/human review.
