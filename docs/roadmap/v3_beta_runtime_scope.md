# v3.0 Beta Runtime Scope

v3.0.0 is the first beta intended for supervised real technical tasks.

## In scope

- terminal-first beta runtime;
- Claude Code adapter;
- Codex adapter;
- Ollama adapter;
- model/token usage ledger;
- command proposal queue;
- approved command execution;
- evidence capture;
- self-review;
- Git stage/commit/push gates;
- teachback closure.

## Out of scope

- full autonomy;
- background agents;
- uncontrolled shell;
- automatic private context access;
- automatic push;
- UI polish;
- marketplace;
- multi-user runtime.

## Post-v3 policy

v3.1+ should be patches, optimization, and improvements from real task evidence.
