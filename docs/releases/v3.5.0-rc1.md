# v3.5.0-RC1 — Conversational Hokage Useful Product

## Objective

Validate Konoha as a conversational, local-first and supervised product for a
real repository audit and controlled documentation patch.

## Included

- Natural-language mission intake.
- Mission Charter and immutable approvals.
- Deterministic checks before local-model invocation.
- One-use Ollama session grant.
- Explicit model/validated/suppressed issue separation.
- Exact patch preview, changed paths and SHA-256.
- Controlled documentation patch application.
- Post-patch regression tests.
- Human review, Teachback, closure and private memory.
- Returning-session continuity.

## Safety

- No external network.
- No model download.
- No arbitrary shell.
- No hidden background work.
- No automatic Git stage, commit, tag, push or release.
- Model output and memory are evidence only.

## Release status

This is a release candidate. Tagging and pushing remain separate future gates.
