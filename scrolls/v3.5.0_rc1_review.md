# v3.5.0-RC1 Council Review

## Decision target

Determine whether the Conversational Hokage is ready for final v3.5.0
hardening and release.

## Required evidence

- Canonical `konoha` mission completes.
- Ollama provider is real, local and explicitly granted.
- Deterministic checks precede model use.
- Suggested, validated and suppressed findings are distinct.
- Patch preview and SHA-256 precede mutation.
- Only approved paths change.
- Post-patch tests pass.
- Review, Teachback and closure pass.
- Private memory is written under ignored paths.
- Git remains untouched until separate approval.

## Known limitation

RC1 supports the existing bounded documentation patch operations. General code
generation and arbitrary repository mutation are out of scope.
