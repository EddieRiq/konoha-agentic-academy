# Konoha Beta Runtime Review Scroll

Use this review Scroll before accepting a v3.0 beta mission result.

## Review questions

1. Was the mission charter created?
2. Did Konoha classify the task correctly?
3. Did Konoha ask for permission before invoking Claude, Codex, Ollama, or commands?
4. Were command proposals reviewed before execution?
5. Were command results captured as evidence?
6. Were token records provider-reported or explicitly estimated?
7. Did Konoha self-review its own performance?
8. Were optimization suggestions evidence-only?
9. Were Git stage, commit, and push separately approved?
10. Was teachback completed before closure?

## Non-authority reminders

- Model output is evidence only.
- Command output is evidence only.
- Self-review is evidence only.
- Git plans are not permission.
- Memory supports action but does not authorize action.
