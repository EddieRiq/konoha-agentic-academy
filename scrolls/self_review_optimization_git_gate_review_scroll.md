# Self-Review, Optimization and Git Operation Gate Review Scroll

Use this Scroll to review v2.9 self-review, optimization, and Git gate outputs.

## Review checklist

- Confirm self-review reports are evidence only.
- Confirm optimization plans are proposal only.
- Confirm token summaries clearly separate actual and estimated usage.
- Confirm Git plans are not treated as permission.
- Confirm the diff was reviewed before stage.
- Confirm tests and audit commands were run before commit.
- Confirm push has explicit remote, branch, token, and `--allow-network`.
- Confirm denied paths are not included.
- Confirm private context stays local.
- Confirm no mission closes from self-review or Git reports.

## Non-authority rule

```text
A good self-review is not permission.
A good optimization plan is not permission.
A clean Git plan is not permission.
```

Human approval remains required.
