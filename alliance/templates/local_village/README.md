# Local Village Template

This folder documents the public-safe pattern for private Local Villages.

A real Local Village should normally live in an ignored local path, for example:

```text
alliance/kirigakure/
```

or in a local path outside the public repository.

Suggested private structure:

```text
config/
memory/vault/
assets/
style/
missions/
context/
local_kage/
reports/
.konoha.local/
```

Public templates must remain generic.
Private context, private memory, private assets, credentials, emails, and company-specific data must not be committed.

Local Village configuration is evidence only and does not authorize execution.
