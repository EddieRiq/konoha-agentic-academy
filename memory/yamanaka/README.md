# Yamanaka Memory

Yamanaka Memory is Konoha's Obsidian-compatible Markdown memory convention.

v2.0.0 provides a minimal memory structure for mission closure:

```text
memory/vault/
  10-missions/
  20-decisions/
  30-tactics/
  40-failures/
  60-context-packs/
```

Mission closure writes only to an explicit `--memory-root`.

Memory supports action, but it does not authorize action.
